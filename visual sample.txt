1. Visuals in Power BI
Visual 1: Total Number of Users (KPI Card)

Visual: Card
Measure (DAX):

Total Users = DISTINCTCOUNT(DIM_USER[USER_KEY])


Slicers:

DIM_DATE[Date]

DIM_USER[Plan Type]

DIM_USER[Region]

Visual 2: Average Meeting Duration by Type & Category

Visual: Clustered Bar Chart (Horizontal)
Axis: DIM_MEETING[MEETING_TYPE]
Legend: DIM_MEETING[MEETING_CATEGORY]
Values:

Avg Duration (Minutes) = AVERAGE(FACT_MEETING_ACTIVITY[DURATION_MINUTES])
Avg Duration (Hours) = DIVIDE([Avg Duration (Minutes)], 60)


Drilldown: Meeting Type → Category
Tooltip: Participant Count (Distinct Users per Meeting)

Visual 3: Users by Meeting Topic (Treemap)

Visual: Treemap
Group: FACT_MEETING_ACTIVITY[MEETING_TOPIC]
Values:

Users per Topic = DISTINCTCOUNT(FACT_MEETING_ACTIVITY[USER_KEY])


Ranking:

Topic Rank = 
RANKX(
    ALL(FACT_MEETING_ACTIVITY[MEETING_TOPIC]),
    [Users per Topic],
    ,
    DESC
)


Top N controlled via What-If Parameter.

Visual 4: Meetings per User

Visual: Bar Chart
Axis: DIM_USER[USER_NAME]
Values:

Meetings per User = DISTINCTCOUNT(FACT_MEETING_ACTIVITY[MEETING_ACTIVITY_ID])


Engagement Classification:

User Engagement Level =
SWITCH(
    TRUE(),
    [Meetings per User] >= 20, "High",
    [Meetings per User] >= 10, "Medium",
    "Low"
)


Used as legend or slicer.

Visual 5: Feature Usage Distribution

Visual: Pie Chart + Table

Measures:

Total Feature Usage = SUM(FACT_FEATURE_USAGE[USAGE_COUNT])

Usage % =
DIVIDE(
    [Total Feature Usage],
    CALCULATE([Total Feature Usage], ALL(DIM_FEATURE))
)

Feature Adoption Rate =
DIVIDE(
    DISTINCTCOUNT(FACT_FEATURE_USAGE[USER_KEY]),
    CALCULATE(DISTINCTCOUNT(FACT_FEATURE_USAGE[USER_KEY]), ALL(DIM_FEATURE))
)


Toggle using Field Parameters (Usage Count vs Adoption Rate).

Visual 6: Meeting Trends Over Time (Dual Axis)

Visual: Line Chart with Secondary Axis

Daily Meeting Count = DISTINCTCOUNT(FACT_MEETING_ACTIVITY[MEETING_ACTIVITY_ID])
Daily Avg Duration = AVERAGE(FACT_MEETING_ACTIVITY[DURATION_MINUTES])

7 Day Moving Avg =
AVERAGEX(
    DATESINPERIOD(DIM_DATE[Date], MAX(DIM_DATE[Date]), -7, DAY),
    [Daily Avg Duration]
)


Date Granularity via Field Parameter: Day / Week / Month.

Visual 7: User Engagement Heatmap

Visual: Matrix with Conditional Formatting
Rows: DIM_DATE[Day Name]
Columns: Hour
Values:

Meeting Intensity = COUNT(FACT_MEETING_ACTIVITY[MEETING_ACTIVITY_ID])


Peak Flag:

Peak Usage =
IF(
    [Meeting Intensity] >=
    PERCENTILEX.INC(
        ALL(FACT_MEETING_ACTIVITY),
        [Meeting Intensity],
        0.8
    ),
    "Peak",
    "Normal"
)


Color scale via conditional formatting.
